# Java Programming Practice & Interview Preparation

## About Java

### What is Java?

Java is a high-level, object-oriented, platform-independent programming language developed by Sun Microsystems (now Oracle). It follows the principle:

**"Write Once, Run Anywhere (WORA)"**

Java source code is compiled into bytecode, which runs on the Java Virtual Machine (JVM).

### Features of Java

* Object-Oriented
* Platform Independent
* Simple and Easy to Learn
* Secure
* Robust
* Portable
* Multithreaded
* Distributed
* High Performance (through JIT Compiler)
* Automatic Memory Management (Garbage Collection)

### Advantages of Java

* Platform independence
* Strong community support
* Rich API and libraries
* Secure execution environment
* Excellent for enterprise applications
* Supports multithreading
* Widely used in MNCs and large-scale applications

### Disadvantages of Java

* More memory consumption than C/C++
* Slower than native compiled languages
* Verbose syntax
* GUI development is less popular compared to modern frameworks

# Java Architecture

### JVM (Java Virtual Machine)

Responsible for executing Java bytecode.

### JRE (Java Runtime Environment)

Provides libraries and JVM required to run Java applications.

### JDK (Java Development Kit)

Provides tools required to develop, compile, and run Java applications.

Relationship:

JDK → JRE → JVM

# Object-Oriented Programming (OOP)

## Class

A blueprint used to create objects.

### Example

Student, Employee, Customer

---

## Object

An instance of a class.

### Example

A specific student record created from the Student class.

---

## Constructor

A special method used to initialize objects.

### Types

* Default Constructor
* Parameterized Constructor

---

## Inheritance

### Definition

Inheritance allows one class to acquire properties and behaviors of another class.

### Benefits

* Code Reusability
* Extensibility

### Types

* Single
* Multilevel
* Hierarchical

---

## Polymorphism

### Definition

One interface, multiple forms.

### Types

## Encapsulation

### Definition
